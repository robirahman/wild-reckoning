# Wild Reckoning

A choose-your-own-adventure wildlife survival simulator. Play as a white-tailed deer in northern Minnesota, navigating predators, parasites, weather, and the struggle to reproduce before you die.

Built with React + TypeScript + Vite + Zustand.

## Development

```bash
npm install
npm run dev
```
